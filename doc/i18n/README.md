# REVELation Submodule Doc Translations

This directory contains translated versions of documentation files that live in `revelation/doc/`.

## Structure

- One folder per language code (`es`, `fr`, etc.)
- Keep translated filenames aligned with source filenames in `revelation/doc/`
- Preserve markdown structure and slide breaks (`---`) from source docs
