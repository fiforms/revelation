# REVELation Presentations

This folder contains presentations created with **REVELation Snapshot Presenter**.

## Viewing and Editing

Download and install the latest REVELation software from:
**[snapshots.vrbm.org/revelation-snapshot-presenter/](https://snapshots.vrbm.org/revelation-snapshot-presenter/)**

Use the software to view, edit, and present your slides.

## Cloud Synchronization

This folder can be synchronized to cloud storage (Google Drive, Dropbox, OneDrive, etc.).

**Important:** Exclude `_media/index.json` from sync. Syncing this file can cause errors on some platforms. You may need to configure your cloud storage to skip this specific file.

## Multi-User Access

⚠️ **Note:** There is currently no file locking mechanism. If multiple instances of the software are running and editing presentations simultaneously, files may be overwritten. True multi-user support is planned for a future version.

Coordinate edits among collaborators to avoid conflicts.
